##📡 Local Wi-Fi Chat

Простой и безопасный чат для локальной сети (Wi-Fi).
Никакого интернета — только ваши устройства и подключение к одной сети.

## ✨ Возможности

- 🔒 **Работает без интернета** — только локальная сеть (Wi-Fi)
- 💬 **Real-time чат** — сообщения приходят мгновенно
- 👥 **Много пользователей** — все, кто подключены к Wi-Fi
- 🖥️ **Не требует установки** — клиент работает в браузере, на ноутбуке, телефоне и других устройствах
- 🐳 **Docker-ready** — запуск одной командой
- 📱 **Mobile friendly** — адаптирован для телефонов

## 🎯 Для чего это

- Общение в поездах, самолётах, на даче (где нет интернета)
- Чат внутри офиса / мероприятия без нагрузки на Wi-Fi
- Учебный проект для понимания WebSocket и сетей
- Ну или чтобы на парах скучно не было


## 🚀 Быстрый старт 
```
git clone https://github.com/ваш-username/local-wifi-chat.git
cd local-wifi-chat
python backend/server.py
python -m http.server 8080
```
